# User Object



## Get Current User
`GET` /user/@me

## Get User
`GET` /user/:user_id

## Report User
`POST` /user/:user_id/report

<br>

### JSON Params
| field    | type   | description                            | default |
|----------|--------|----------------------------------------|---------|
| category | string | One category from category list        | Other   |
| reason   | string | Report's reason. Minimum 10 characters | -       |

<br>

### Category choices
- Advertising
- Spamming
- Raiding
- Harassing
- Other
